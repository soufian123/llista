# Lista de Compra - Proyecto de Python

Este proyecto de Python es una aplicación simple para gestionar una lista de compra.
Permite a los usuarios agregar categorías, artículos y registros a la lista de compra,
y luego guardar la lista en un archivo JSON para su posterior recuperación.


## Funcionalidades Principales

- Creación, lectura, actualización y eliminación de categorías.
- Creación, lectura, actualización y eliminación de artículos.
- Creación y lectura de registros que asocian artículos con cantidades en la lista de compra.
- Guardar la lista de compra en un archivo JSON.
- Cargar la lista de compra desde un archivo JSON.
